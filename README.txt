
Este repositório contém o projeto oriundo do curso Desenvolvimento para Internet e Banco de 
Dados com Python e Django, ministrado por Rafael Galleani, na plataforma Digital Innovation one.
O projeto é a implementação de uma agenda usando a linguagem python e framework django. Especificamente,
encontram-se desenvolvidos nesse projeto:

- banco de dados sqlite;
- tela de login;
- cadastramento de novos usuários;
- campos de cadastro: título do evendo a ser agendado, data e horário, breve descrição do evento;
- visualização com o básico de html.
